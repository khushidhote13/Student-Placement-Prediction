Student Placement Prediction

##  Project Overview

This project predicts whether a student will be placed based on academic performance and other features using multiple Machine Learning classification algorithms.

## Features

- Data Preprocessing
- Data Visualization
- Label Encoding
- Feature Scaling
- Model Training
- Model Comparison
- Prediction

## Machine Learning Models Used

- Logistic Regression
- Decision Tree
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Random Forest
- Gradient Boosting

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset

Student Placement Dataset (`placementdata.csv`)

## Evaluation

The models were evaluated using:
- Accuracy Score
- Classification Report
- Confusion Matrix

##  How to Run

1. Clone the repository
2. Install the required libraries
3. Open the notebook
4. Run all cells
